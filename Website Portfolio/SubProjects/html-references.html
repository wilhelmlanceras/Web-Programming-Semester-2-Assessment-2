HTML Tags

1. <title>

Appears on the browser tab.

<title>Page Title</title>

2. <meta>

Provides metadata about the page, such as character encoding.

<meta charset="UTF-8">

3. <h1> to <h6>

Define heading levels, with <h1> as the largest.

<h1>Main Heading</h1>

4. <p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

5. <br>

Inserts a line break without starting a new paragraph.

<br>

6. <dl>

Creates a description list, with terms (<dt>) and descriptions (<dd>).

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
</dl>

7. <ul> and <ol>

Define unordered (bulleted) and ordered (numbered) lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

8. <code> and <pre>

Used for displaying code with proper formatting.

<pre>
<code>console.log("Hello");</code>
</pre>

9. <img>

Embeds an image on the webpage.

<img src="image.jpg" alt="Description">

10. <div>

Defines a container for grouping elements.

<div>
  <h2>Section Heading</h2>
  <p>This is inside a div.</p>
</div>

Common CSS Properties

1. background-color

Sets the background color of an element.

div {
  background-color: #f4f4f4;
}

2. color

Changes the text color.

p {
  color: blue;
}

3. font-family

Sets the font style.

body {
  font-family: Arial, sans-serif;
}

4. font-size

Defines the text size.

h1 {
  font-size: 2em;
}

5. font-style

Applies italic or other font styles.

p {
  font-style: italic;
}

6. font-weight

Controls text thickness (e.g., bold).

h2 {
  font-weight: bold;
}

7. line-height

Adjusts the spacing between text lines.

p {
  line-height: 1.6;
}

8. margin

Controls the space outside an element.

div {
  margin: 20px;
}

9. text-align

Aligns text inside an element.

h1 {
  text-align: center;
}

10. text-decoration

Modifies text appearance (e.g., underline, none).

a {
  text-decoration: none;
}
